uint32_t GetRanNum ( int  n)

주어진 비트 수에 따라 랜덤 숫자를 생성

주어진 비트 수에 따라 랜덤 숫자를 생성하는 함수

이 함수는 입력으로 받은 비트 수(n)에 따라 16비트 또는 32비트 범위의 랜덤 숫자를 생성하여 반환 랜덤 시드 생성기를 사용하여 매 호출마다 새로운 랜덤 값을 생성

Parameters
n생성할 랜덤 숫자의 비트 수. 16 또는 32만 지원
Returns
uint32_t 생성된 랜덤 숫자
  • n이 32일 경우: 0부터 2^32-1(0xFFFFFFFF) 범위의 랜덤 숫자
  • n이 16일 경우: 0부터 65535(2^16-1) 범위의 랜덤 숫자
  • n이 16 또는 32가 아닐 경우: 0을 반환
Note
  • std::random_device는 랜덤 시드 값을 생성하는 데 사용
  • std::mt19937는 Mersenne Twister 알고리즘을 기반으로 한 난수 생성기
  • std::uniform_int_distribution은 지정된 범위 내에서 균등 분포를 가진 난수를 생성

◆ GetServerIP()

std::string GetServerIP ( )

현재 서버의 IP 주소 반환

현재 서버의 IP 주소를 반환하는 함수

이 함수는 시스템의 호스트 이름을 기반으로 서버의 IP 주소를 조회하여 반환 반환된 IP 주소는 IPv4 형식의 문자열로 제공

Returns
std::string 서버의 IP 주소
  • 성공 시: IPv4 형식의 IP 주소 문자열
  • 실패 시: 빈 문자열 ("")
Note
  • gethostname 함수는 시스템의 호스트 이름을 조회
  • gethostbyname 함수는 호스트 이름을 기반으로 IP 주소 정보를 조회
  • inet_ntoa 함수는 이진 형식의 IP 주소를 사람이 읽을 수 있는 문자열 형식으로 변환
Warning
  • 이 함수는 IPv6 주소를 지원하지 않습니다. IPv4 주소만 반환
  • gethostbyname는 오래된 함수로, 최신 시스템에서는 getaddrinfo 사용을 권장
  • 함수 호출 중 오류가 발생하면 perror를 통해 에러 메시지가 출력
