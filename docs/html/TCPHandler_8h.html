Detailed Description

TCP 연결 및 RTSP 메시지 처리를 위한 싱글톤 클래스 헤더

TCP 소켓을 생성하고 클라이언트 연결을 관리하며, RTSP 프로토콜에 따른 메시지 송수신을 처리하는 기능을 제공하는 클래스

  • TCP 소켓 생성 및 초기화
  • 클라이언트 연결 수락 및 관리
  • RTSP 요청/응답 메시지 처리
