Detailed Description

H264 비디오 프레임을 처리하는 클래스

H264 포맷의 비디오 파일을 읽고 NAL 단위로 프레임을 추출하는 기능을 제공하는 클래스. mmap을 사용하여 파일을 메모리에 매핑하고, start code를 찾아 NAL 단위로 프레임을 파싱한다.
