Detailed Description

RTSP 서버 메인 클래스 헤더

RTSP 프로토콜을 지원하는 미디어 스트리밍 서버의 핵심 기능을 제공하는 클래스

  • 싱글톤 패턴을 사용한 서버 인스턴스 관리
  • 클라이언트 연결 및 세션 관리
  • 프로토콜 타입(H264/Opus) 관리
