\doxysubsection{Detailed Description}
RTP 패킷 관리를 위한 클래스 

RTP 패킷을 생성하고 전송하는 클래스

실제 구현은 RTPPacket.\+hpp에 정의되어 있으며, RTP 패킷의 생성, 관리, 전송 클래스

RTP Packet = Header + Payload 로 구성된 패킷을 생성하고 전송하는 기능을 제공한다.
